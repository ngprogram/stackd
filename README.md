Stat

## Development

### Setup

Install mocha

```
npm install -g mocha nodemon
```

Install all packages and bower components

```
npm install
bower install
```

### Running

Running for development (using nodemon):
```
npm run start-dev
```
Running on production:
```
npm start
```
### Testing

```
npm test
```
